# steno-dictionaries

My personal dictionaries for Plover.

Assumed plugins:

- Translation providers:
  - `plover-python-dictionary`
  - `plover-dict-commands`
  - `plover-stitching`
  - `plover-emoji`
  - `plover-number-format`
- UI:
  - `plover-clippy`
  - `spectra-lexer`

Dictionary ordering:

- `user.json`
- `emoticons.json`
- `plugin-integrations.json`
- `uni-number-reversals.json`
- `emily-modifiers.py`
- `emily-symbols.py`
- `commands.json`
- `main.json`
